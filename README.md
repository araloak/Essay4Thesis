# Essay2Thesis

## 项目简介
Essay2Thesis 是一个旨在帮助`AI方向`的硕士、博士将发表在英文学术会议（比如AAAI、NeurIPS等）上的小论文（essay）内容按照国内计算机领域博士毕业论文的写作风格重新整理的工具。该项目通过自动化的方式生成符合博士论文写作规范的内容，帮助用户高效完成论文撰写。

**注意：本项目仅供学习与研究使用，禁止用于任何形式的学术不端行为。**

---

## 功能概述
### 已实现功能

- **论文章节生成**：
  - 支持生成essay对应在毕业论文里相应章节的前言、引言、方法。
  - 加载用户提供的essay内容和毕业论文写作样例进行参考。
  - 提供初版提示词进行参考
- **多模型支持**：
  - 自定义使用模型API，支持从不同模型生成的多个候选自动挑选最佳内容
- **输出格式**：
  - 以latex格式保存在txt文件中

## 工作流程与逻辑

### 用户输入

1. **Essay 内容**：
   - 用户需提供英文 essay 的各部分内容（如 `intro.txt`、`abs.txt`）。
   - 文件路径示例：`data/essays/essay1/intro.txt`。
2. **博士论文写作样例**：
   - 用户需提供符合博士论文写作风格的样例内容（如 `example_essay4thesis_引言.txt`）。
   - 文件路径示例：`data/thesis/example_essay4thesis_引言.txt`。
3. **提示词**：
   - 系统提示词和章节写作提示词需存放在 `prompts` 文件夹中。
   - 文件路径示例：`prompts/sys_prompt1.txt`、`prompts/essay4thesis_引言_prompt.txt`。

### 工作流程

1. **加载提示词与参考内容**：
   - 代码从 `prompts` 文件夹中加载系统提示词（如 `sys_prompt1.txt`）和章节写作提示词（如 `essay4thesis_引言_prompt.txt`）。
   - 在 `data/essays` 和 `data/thesis` 文件夹中加载待处理的essay内容和博士论文写作样例（需要自行准备）。
2. **生成章节内容**：
   - 根据加载的提示词和参考内容，调用指定的语言模型（如 `dsr1`、`gemini-2.0-flash` 等）生成章节内容。
   - 如果模型支持思维链输出（如 `dsr1`），会额外保存带有思维链的版本。
3. **候选内容对比**：
   - 对多个模型生成的候选内容进行对比，挑选出最佳内容。
   - 使用 `best_of_N_candidates` 方法实现候选内容的对比与选择。
4. **保存输出**：
   - 将生成的内容保存到 `data/thesis/第三章` 文件夹中对应的章节目录下。

### 逻辑概述

- **模块化设计**：

  - 每一节的生成逻辑独立封装在对应的 Python 文件中（如 `essay4thesis_intro.py`、`essay4thesis_abs.py`）。
  - 公共功能（如加载提示词、保存文件）封装在 `util.py` 中。

  - 模型API配置保存在`config.json`中
  - 直接运行`essay4thesis_abs.py`进行章节前言写作，最终结果保存在`/data/thesis/第三章/前言/最佳.txt`。其余内容写作以此类推
