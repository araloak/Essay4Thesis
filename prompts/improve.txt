## 修订任务
根据评阅意见对原始博士论文草稿进行修订，严格遵循以下规则：
- 关键术语用词和特殊名称与博士论文前言中的说法保持一致；
- 仅在批评意见中明确指出需要修改的地方进行改写，其他未被指出的内容不变；
- 不引入与essay对应内容或原稿无关的事实或观点；不得引入新的未经提供的技术细节或实验数据；
- 保持论文的 LaTeX 风格和格式（例如公式、图表引用的 LaTeX 标记应保留或按建议修正）；
- 对于需改写的句子，直接在稿件中替换为改写后的句子，保持段落和小节结构不变；
- 输出应为完整的修订后论文片段文本（即整段/整节内容），不要附加额外的解释性文字或评注。

## 格式要求
- 必须以纯 JSON 格式输出（不要在前后添加任何 Markdown、解释或文本）。
- JSON 字段定义如下：
	- `revised_text`: 字符串，完整的修订后论文片段（保持 LaTeX 格式），该字段为必填。

示例输出：
```
{
		"revised_text": "...完整修订后文本..."
}
```
